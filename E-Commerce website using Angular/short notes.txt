#<router-outlet> use
-Router-outlet in Angular works as a placeholder which is used to load the different components 
dynamically based on the activated component or current route state.

#subscribe() function
-we subscribe to that function if we have any function we want to observe on an Angular application. 
Whenever any change is made on that function, we can see the change using the subscribe() method.
